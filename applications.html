Positioning

Applications

DSI audits a supplied AI answer against a defined set of expected points and reports what was omitted, with source evidence and a reproducible fingerprint.

Example application

AI summary assurance.

Compare a generated summary against a defined set of expected points and report the omissions — each with source evidence and a reproducible fingerprint. The same input produces the same report.

Also supported

Advisory AI.

Audit an advisory model answer against a configured set of expected points and report which were surfaced, which were omitted, and the source evidence. Example domains include career, finance, and relationship decision-support.

Current

Important scope note

Scope

These examples describe potential areas of exploration. The current DSI implementation focuses on expected-path visibility in advisory model outputs.

This work measures visibility of configured expected paths in model outputs. It does not measure advice quality, factual correctness, user outcomes, or regulatory compliance.

DSI is a local, stateless assurance sidecar. It audits a supplied response; it does not generate one. It measures configured expected-path visibility — not advice quality, safety, or compliance.
